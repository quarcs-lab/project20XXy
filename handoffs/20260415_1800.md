## Decisions Made

- `{{< include >}}` for tables, `{{< embed >}}` for figures (Quarto embed limitation)
- Two-pass render required (first pass generates table .md files, second includes them)
- Manual pipe-delimited markdown for regression tables (etable/esttab md output doesn't render)
- 6×4 inch figure standard across all languages
- LaTeX-to-GFM conversion for LLM markdown (cleaner than HTML-to-markdown)
- Quarto `llms-txt` feature is website-only, not available for manuscript projects
- GitHub Pages served from gh-pages branch with _manuscript/ contents
