## Key Decisions

| Decision | Choice | Rationale |
| --- | --- | --- |
| Python version | 3.12 | User preference for recent version |
| Package manager | uv + pyproject.toml | Matches reference repo (quarcs-lab/project2025s), fast and reproducible |
| Output formats | HTML + PDF + Word | Simplified from reference (which had 2 PDFs + JATS + MECA) |
| Notebook pairing | Jupytext (.ipynb + .md:myst) | Clean git diffs, matches reference repo |
| PDF engine | LuaLaTeX via KOMA-Script (scrartcl) | Standard academic layout, uses existing TeX Live |
| LaTeX template | Kept in templates/ | User wants it as an alternative to Quarto |
| Notebook kernels | Python + R + Stata | User needs all three for different analyses |
